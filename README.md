# 幼儿园圣诞树生成器

一个基于Python Flask的幼儿园圣诞树生成系统，包含管理员登录和圣诞树生成功能。

## 功能特性

1. **管理员登录**：使用指定账号（test）和密码（admin）登录
2. **圣诞树生成**：输入小朋友姓名，生成专属圣诞树
3. **数据保存**：生成的圣诞树会自动保存到数据库
4. **数据调取**：再次输入相同姓名时，会展示已保存的圣诞树

## 技术栈

- Python 3.x
- Flask 2.3.3
- Flask-SQLAlchemy 3.1.1
- SQLite 数据库

## 安装与运行

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python app.py
```

### 3. 访问应用

打开浏览器，访问：http://localhost:5000

## 使用流程

1. **登录**：输入用户名 `test` 和密码 `admin`
2. **生成圣诞树**：在仪表板输入小朋友姓名，点击"生成圣诞树"按钮
3. **查看结果**：系统会显示生成的圣诞树，并保存到数据库
4. **再次访问**：下次输入相同姓名时，系统会展示已保存的圣诞树

## 数据库结构

### ChristmasTree 表

| 字段名       | 类型        | 说明           |
|-------------|-------------|---------------|
| id          | Integer     | 主键，自增     |
| child_name  | String(100) | 小朋友姓名     |
| tree_data   | Text        | 圣诞树数据     |

## 开发说明

- 数据库使用SQLite，无需额外配置
- 项目使用Flask框架，支持热重载
- 页面设计采用童趣化风格，适合幼儿园使用
